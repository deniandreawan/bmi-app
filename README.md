### How to run

1. Clone the repository

   ```bash
   git clone https://github.com/deniandreawan/bmi-app.git
   ```

2. Install the dependencies

   ```bash
   cd bmi-app
   yarn install
   ```

3. Run the application
   ```bash
   yarn start
   ```
